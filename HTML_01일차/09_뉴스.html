갤럭시Z폴드3에서도 '태스크바'…삼성, 폴더블∙워치 업데이트 시작


| 플립·폴드 전 기종·갤럭시 워치4·워치4 클래식 대상 업데이트

[데일리안 = 남궁경 기자] 삼성전자가 기존 갤럭시 폴더블 스마트폰과 워치 사용자들을 위해 '원 UI 4.1.1'과 '원 UI 워치4.5' 등 소프트웨어 업데이트를 5일 시작한다.


업데이트 시 기존 갤럭시 폴더블 스마트폰과 워치 사용자는 새롭게 출시된 갤럭시 Z 플립4∙폴드4와 워치5에 도입된 기능들을 이용할 수 있다.


원 UI 최신 업데이트는 보다 향상된 사용성과 생산성, 고객 맞춤형 기능 등에 초점을 맞췄다.


원 UI 4.1.1 업데이트는 이날 국내를 시작으로 북미, 유럽 등으로 순차 확대될 예정이다.


지원 단말은 갤럭시 Z 플립3∙폴드3 사용자를 시작으로 1세대 갤럭시 플립과 폴드까지 순차적으로 확대 적용된다.


이를 통해 기존 모델을 사용하는 소비자들도 태스크바와 같이 개선된 멀티테스킹과 플렉스캠 기능을 경험할 수 있다.


갤럭시 워치4와 워치4 클래식에서는 원 UI 워치 4.5 소프트웨어가 적용된다. 이를 통해 신규 워치 페이스와 향상된 맞춤 설정 기능, 쿼티 키보드 기반의 편리한 타이핑 경험 등을 제공한다.


한편, 갤럭시 워치3와 워치 액티브2에도 소프트웨어 업데이트를 통해 신규 워치 페이스 2종, 수면 중 코골이 녹음 기능이 추가된다.


삼성전자는 관계자는 "갤럭시 폴더블 스마트폰과 워치 사용자들이 보다 혁신적인 최신 모바일 경험을 지속적으로 누릴 수 있도록 앞으로도 지원을 아끼지 않겠다"고 말했다.
