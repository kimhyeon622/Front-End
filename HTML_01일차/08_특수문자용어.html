특수문자 용어정리

` : 백틱(Backtick), 그레이브(Grave)

~ : 물결, 틸드(Tilde)

! : 느낌표, 엑스클러메이션(Exclamation mark)

@ : 골뱅이, 앳(At sign)

# : 우물정, 샵(Sharp), 넘버(Number sign)

$ : 달러(Dollar sign)

% : 퍼센트(percent sign)

^ : 캐럿(Caret)

& : 앤드, 엠퍼센드(Ampersand)

* : 별표, 에스터리스크(Asterisk)

- : 짝대기, 바, 마이너스, 하이픈(Hyphen), 대시(Dash)

_ : 밑줄, 언더바, 언더스코어(Underscore), 로대시(LowDash)

= : 같다, 이퀄(Equals sign)

+ : 더하기, 플러스(Plus sign)

" : 큰따옴표, 더블쿼터, 쿼테이션(Quotation mark)

' : 작은따옴표, 싱글쿼터, 아포스트로피(Apostrohe)

: : 떙떙, 콜론(Colon)

; : 세미콜론(Semicolon)

, : 쉼표, 콤마(Comma)

. : 점, 닷(Dot), 피리어드(Period)

? : 물음표, 퀘스천(Question mark)

/ : 슬래시(Slash)

| : 버티퀄바(Vertical bar)

\ : 백슬래시(Backslash), 역슬래시

() : 괄호, 소괄호, 퍼렌서시스(Parenthesis)

{} : 중괄호, 브레이스(Brace)

[] : 대괄호, 브래킷(Bracket)

> : 꺽쇠괄호, 앵글브래킷(Angle Brecket)
